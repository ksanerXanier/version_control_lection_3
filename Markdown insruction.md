## Списки

Чтобы добавить ненумерованные списоки, необходимо пункты выделить звёздочкой (*). Например, вот так:
 * Элемент 1
 * Элемент 2
 * Элемент 3

 Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. Напимер, вот так:
 1. Первый пункт
 2. Второй пенкт
 3. Третий пункт
